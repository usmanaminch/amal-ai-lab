Learning Journal

Day 1 — Starting my personal website

Today I created the first version of my personal website. I learned that a website can have tabs for different parts of my life and projects. I also learned that GitHub saves code like a project notebook.

Day 2 — Adding a Favorite Things section

Today I added Favorite Things and learned that HTML creates content, CSS controls colors and layout, and JavaScript makes tabs and buttons work.

Day 3 — Human Body Explorer

Today I built Human Body Explorer for my Future Surgeon Lab. It helps me understand human body parts like the brain, heart, lungs, bones, muscles, skin, teeth, and jaw. I learned that CSS controls the colors, and JavaScript makes Explore buttons and filters work. I also learned why safety notes matter: this is for learning, not medical advice.

Day 3 Concept Answers

  1. The project helps me understand the body parts of humans.
  2. The brain was interesting because it helps us generate thoughts and control the body.
  3. CSS controls the colors and darker style.
  4. JavaScript makes the Explore buttons and filters work.
  5. The safety note shows that I am not a real doctor and this is just for learning.

Day 4 — Clickable Human Body Diagram

Today I upgraded Human Body Explorer so the actual body parts are clickable. The heart can pump blood, the lungs can breathe, and more than one body part can stay active at the same time. I learned that JavaScript can remember active body parts and CSS can animate them.

Day 4 Concept Answers

  1. HTML, CSS, and JavaScript worked together to make the page interactive.
  2. JavaScript detects clicks and turns animations on or off.
  3. Clicking the actual body part feels more real than clicking a side button.
  4. JavaScript remembers the active body parts, like heart and lungs.
  5. The safety note shows this is for learning, not medical advice.

Day 5 — Plant Care Helper

Today I built Plant Care Helper. It lets a user upload a plant photo, preview the image, choose symptoms, and generate a care plan. I learned that today’s version does not truly understand the photo yet. It uses the choices selected by the user to make helpful recommendations.

Day 5 Concept Answers

  1. The upload button lets the user put a plant picture into Plant Care Helper.
  2. The website uses the choices selected to help make a care plan.
  3. CSS makes the page colorful and cute.
  4. JavaScript reads the choices and makes the care plan for the plant.
  5. The safety note shows this is educational and not advice from a botanist.

Day 6 — AI-Assisted Plant Care + Outfit Organizer

Today I upgraded Plant Care Helper so it can create an AI vision prompt, accept a pasted AI answer, and turn that answer into a cute plant care report. I also built Outfit Organizer, a fashion magazine-style project that uses a closet photo and style interview questions to create a 7-day outfit lookbook.

Day 6 Concept Notes

  1. Plant Care Helper uses an AI-assisted workflow: photo, prompt, AI answer, then report.
  2. Outfit Organizer asks style questions because good recommendations need preferences.
  3. The closet photo helps AI notice clothing categories, colors, and possible outfit pieces.
  4. The website turns the AI answer into a prettier weekly lookbook.
  5. Safety matters because closet photos should not include faces, addresses, school names, or private details.
