Day 1 — Starting my personal website

Today I created the first version of my personal website. I learned that a website can have tabs for different parts of my life and projects. I also learned that GitHub saves code like a project notebook.

Day 2 — Adding a Favorite Things tab

Today I changed Favorite Things from a card into its own tab. I also learned that HTML creates the tab and content, CSS controls the colors and layout, and JavaScript makes the tabs and reminder button work.

Today’s 5 Concept Questions
