Learning Journal

Day 1 — Starting my personal website

Today I created the first version of my personal website. I learned that a website can have tabs for different parts of my life and projects. I also learned that GitHub saves code like a project notebook.

Day 2 — Adding a Favorite Things section

Today I added Favorite Things and learned that HTML creates content, CSS controls colors and layout, and JavaScript makes tabs and buttons work.

Day 3 — Human Body Explorer

Today I built Human Body Explorer for my Future Surgeon Lab. It helps me understand human body parts like the brain, heart, lungs, bones, muscles, skin, teeth, and jaw. I learned that CSS controls the colors, and JavaScript makes Explore buttons and filters work. I also learned why safety notes matter: this is for learning, not medical advice.

Day 3 Concept Answers

  1. The project helps me understand the body parts of humans.
  2. The brain was interesting because it helps us generate thoughts and control the body.
  3. CSS controls the colors and darker style.
  4. JavaScript makes the Explore buttons and filters work.
  5. The safety note shows that I am not a real doctor and this is just for learning.
