# Book Selector · Day 7 v3 Your Library

A library-style book recommendation app.

## What changed in v2

The layout now feels like a library app instead of a generic prompt app.

It includes:

- Library Card reader profile
- Librarian Desk prompt area
- Card Catalog reader match summary
- Wooden Recommended Shelf
- Book-spine style recommendation cards

## Flow

1. Fill out the library card.
2. Ask the AI librarian.
3. Paste AI recommendations.
4. Add books to the digital shelf.

## Concepts practiced

- user preference forms
- prompt engineering
- recommendation systems
- JavaScript parsing
- app layout and product design

## Safety

The recommendations should be age-appropriate. A parent can check any book first if unsure.
