# Decor Planner · Day 8 v2

A decor-store-style room makeover planning app.

## v2 updates

- Looks more like walking into a decor store
- Storefront hero with awning
- Window display
- Store map
- Photo Drop-Off
- Design Counter
- Decorator Desk
- Sample Wall
- Showroom Plan Board
- Aisle-style decor cards

## What it does

- Upload or paste a room photo
- Answer a design quiz
- Create an AI decorator prompt
- Paste the AI answer
- Turn the answer into showroom decor cards

## Future upgrade

Later, this can connect to AI image generation to create visual room makeover pictures.
