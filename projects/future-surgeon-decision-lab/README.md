# Future Surgeon Procedure Lab · Day 12 v2

An interactive educational surgeon career simulator.

## v2 changes

- Cases now have procedure models.
- Students drag the correct tool onto the body/case model.
- Steps are specific and must be completed in order.
- The model visually changes after each correct tool.
- Quiz unlocks after finishing one case.

## Cases

- Wrist injury: X-ray → splint → cast/support
- Jaw injury: airway check → face X-ray → gauze → jaw support
- Skin repair: sterile wash → suture kit → bandage
- Belly pain: vitals → ultrasound → IV fluids → call surgical team

## Safety

This is educational only and not medical advice.
It does not diagnose, treat, or give real medical instructions.
Real patients need real clinicians.
