# Patient Vitals Monitor Learning Lab · Day 19

A kid-friendly medical learning project about patient vital signs.

## Topics
- Heart rate
- Oxygen level
- Blood pressure
- Breathing rate
- Temperature
- Teamwork and monitoring

## Safety note
This is an educational simulation only. It is not medical advice and does not teach real clinical care.
