# Rocket Model Lab · Day 9 v3

Drag-and-drop builder version.

## v3 changes

- Builder tray now shows loose rocket parts.
- Users drag pieces into matching rocket outlines.
- Instruction book is mostly visual with short words.
- The rocket does not build itself.
- Part Summary stays locked until all selected pieces are dragged into place.
- Launch uses the exact built rocket model.
- AI Coach is still removed.
- Pieces remain middle-school level.

## Safety rule

This is a digital simulation only. It does not teach how to build motors, engines, propellants, or dangerous launch systems.


## v4 visual update

The layout now feels more like walking into Mission Control and building a rocket in a space-center assembly bay.


## v5 cosmic decor

Added a moon, planets, and stars so the layout feels more like a space-center environment.


## v6 mission upgrades

Added glowing control screens, a launch countdown panel, and astronaut / engineer silhouettes for a stronger Mission Control feel.


## v7 animated effects

Added blinking control lights on the control screens and a launch smoke effect on liftoff.
