# Snack Helper · Day 7 v2 Chef's Kitchen

A cozy cooking-show kitchen version of Snack Helper.

## What changed

This version feels like a warm kitchen where a friendly chef mentor helps the student make quick snack ideas.

It includes:

- Chef's Welcome
- Pantry Check
- Chef's Recipe Slip
- Snack Counter
- Recipe-card style snack results
- Strong safety reminders

## Flow

1. Fill out the pantry check.
2. Ask the snack chef.
3. Copy the recipe request into ChatGPT or Gemini.
4. Paste the answer back into the app.
5. Put recipes on the counter.

## Safety

Ask an adult before using knives, stove, oven, blender, or anything risky. Avoid allergy foods.


## v3 update

Made the Snack Counter lighter with a warm butcher-block style color so the recipe cards are easier to read.
