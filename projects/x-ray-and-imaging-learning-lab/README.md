# X-Ray and Imaging Learning Lab · Day 20

A kid-friendly medical learning project about common medical imaging types.

## Topics
- X-ray
- MRI
- CT scan
- Ultrasound
- Matching scan types to learning cases
- Imaging safety basics

## Safety note
This is an educational simulation only. It is not medical advice and does not diagnose real medical images.
